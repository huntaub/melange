Developer Preview

Melange is currently in a developer preview release. This has several implications. If you are not a developer looking to create plugins for Melange, this release is probably not for you. Not all features are expected to work correctly, and not all security guarantees will be met.

Account Restoration

Remember to back up your computer to a secure location while using Melange. If you lose your account information (stored only in your ~/Library/Application Support/Melange folder), there is nothing that we can do to restore your account.
