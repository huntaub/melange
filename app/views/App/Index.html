Melange lets you control your data like never before. Security

Melange runs on the innovative, new AirDispatch protocol. Lightyears more secure than email, AirDispatch allows users to sleep comfortably at night without having to worry about addressing spoofing, encryption, or data leakage.

Flexibility

Melange uses the new Web Plugins Library to add extensibility to the messages that it can receive. Anyone can go online and define a free protocol that specifies exactly what needs to be in a message and how it is displayed in the interface.
