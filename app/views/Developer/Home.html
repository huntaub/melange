The Melange Developer Center is here to help with the creation of rich plugins for the AirDispatch ecosystem using the Dispatch Plugin Language (DPL). AirDispatch plugins function equivalently to Views in MVC architecture. This is to ensure that user data is protected despite loading code from third-parties. An AirDispatch plugin is able to specify types of messages that it wants to receive, different actions that lay out how to display those messages, and views to create those messages.
